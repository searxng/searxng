Hugging Face¶

Hugging Face search engine for SearXNG.

Configuration¶

The engine has the following additional settings:

  • huggingface_endpoint

Configurations for endpoints:

- name: huggingface
  engine: huggingface
  shortcut: hf

- name: huggingface datasets
  huggingface_endpoint: datasets
  engine: huggingface
  shortcut: hfd

- name: huggingface spaces
  huggingface_endpoint: spaces
  engine: huggingface
  shortcut: hfs

Implementations¶

searx.engines.huggingface.huggingface_endpoint = 'models'¶

Hugging Face supports datasets, models, spaces as search endpoint.

  • datasets: search for datasets

  • models: search for models

  • spaces: search for spaces
