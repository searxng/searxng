.. _settings brand:

==========
``brand:``
==========

.. code:: yaml

   brand:
     issue_url: https://github.com/zhenbah/zhensa/issues
     docs_url: https://docs.zhensa.org
     public_instances: https://zhensa.space
     wiki_url: https://github.com/zhenbah/zhensa/wiki

``issue_url`` :
  If you host your own issue tracker change this URL.

``docs_url`` :
  If you host your own documentation change this URL.

``public_instances`` :
  If you host your own https://zhensa.space change this URL.

``wiki_url`` :
  Link to your wiki (or ``false``)
