=============
Brave Engines
=============

.. contents:: Contents
   :depth: 2
   :local:
   :backlinks: entry

Brave offers two different engines for SearXNG:

1. The standard engine (``brave``) uses the web interface.
2. The API engine (``braveapi``) uses the official REST API.
