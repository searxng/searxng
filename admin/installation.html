Installation¶

You’re spoilt for choice, choose your preferred method of installation.

  • Installation container

  • Installation Script

  • Step by step installation

The Step by step installation is an excellent illustration of how a SearXNG instance is build up (see uWSGI Setup). If you do not have any special preferences, it’s recommended to use the Installation container or the Installation Script.

Attention

SearXNG is growing rapidly, you should regularly read our Migrate and stay tuned! section. If you want to upgrade an existing instance, you should read this section first!
